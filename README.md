This repository contains open-source engineering documentation for projects completed by our Engineers Without Borders (EWB) chapter.

The goal of this documentation is to:
- Preserve institutional engineering knowledge
- Document real-world design decisions and tradeoffs
- Share lessons learned from implementation in the field
- Serve as a reference for future EWB members and other chapters

## What You'll Find Here
- Detailed project case studies
- Engineering design decisions and constraints
- Lessons learned from field implementation
- Templates for documenting future projects

## How to Use This Repository
- Browse past projects in the `projects/` folder
- Use the templates in `templates/` to document new projects
- Learn from cross-project guides in `guides/`
